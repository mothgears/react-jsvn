# REACT-JSVN

React render for JSVN - ESNext JSX alternative with class inheritance

#### JSVN
https://www.npmjs.com/package/jsvn

#### Documentation and examples
https://jsvn.dev

#### Versions
For `react@>=18` use `react-jsvn@2.x`  
For `react@<18` use `react-jsvn@1.x`

#### Installation
npm
```
npm i react-jsvn
```

yarn
```
yarn add react-jsvn
```
